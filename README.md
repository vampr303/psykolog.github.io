## 📋 Что можно редактировать в админ-панели

### 1️⃣ Вкладка "Профиль"
- Имя и фамилия психолога
- Специальность (например: "Клинический психолог")
- Биография
- Опыт работы (количество лет)
- Номер телефона
- Username в Telegram

Все изменения сохраняются в базу данных Firebase при нажатии кнопки "Сохранить профиль".

### 2️⃣ Вкладка "Разделы" - ДОБАВЛЕНИЕ НОВЫХ РАЗДЕЛОВ!

**Кнопка "Добавить новый раздел"**

При нажатии вы сможете создать:

#### **Тип 1: "Текст"**
- Простой текстовый раздел
- Например: "О моем подходе", "Методология", "Что я предлагаю"
- Можно добавить длинный текст с описанием

#### **Тип 2: "Карточки"**
- Сетка с элементами
- Пример: "Мои специализации", "Преимущества работы со мной"
- Каждая карточка имеет название и описание
- Можно добавлять неограниченное количество карточек

#### **Тип 3: "Блог"**
- Статьи/посты с датой публикации
- Идеально для влога или блога
- Каждый пост имеет название, содержимое и дату
- Хранятся в хронологическом порядке

**Как работает:**
1. Нажимаем "Добавить новый раздел"
2. Вводим название раздела (например: "Мой блог")
3. Выбираем тип (1, 2 или 3)
4. Редактируем раздел - добавляем элементы
5. Сохраняем

**Пример использования:**
- Создаём раздел "Блог" (тип 3)
- Добавляем статью: "Как справиться с тревогой" (дата + текст)
- Добавляем ещё статьи
- Раздел появляется на сайте со всеми постами!

### 3️⃣ Вкладка "Отзывы"
- Просмотр всех отзывов
- Удаление нежелательных отзывов
- Отзывы появляются из формы на сайте

## ❓ Частые вопросы

**Q: Могу ли я добавить больше услуг?**
A: Услуги редактируются через раздел "Разделы" → создайте раздел "Услуги" (тип "Карточки")

**Q: Как добавить влог?**
A: Новый раздел → тип "Блог" → называете "Влог" → добавляете видео описания и ссылки в содержимое

**Q: Все данные удалятся?**
A: Нет, все хранится в Firebase. Они будут даже если отключить сайт.

**Q: Кто может редактировать?**
A: Только люди с пароль `psykolog2024` через админ-панель.
